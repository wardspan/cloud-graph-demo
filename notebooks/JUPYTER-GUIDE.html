📖 How to Use Jupyter Notebooks

🔑 Most Important Skill: Running Code Cells

  1. Click on a code cell (gray background) - it gets a blue border
  2. Press Shift + Enter - this runs the code
  3. Wait for results - output appears below the cell
  4. Move to next cell - repeat the process

📋 Essential Rules

  • Always go top to bottom - Cells depend on previous cells
  • Read text cells first - They explain what's coming next
  • Run code cells in order - Skipping causes errors
  • When stuck, restart - Kernel → Restart Kernel

🆘 Troubleshooting

Common Issues & Solutions:

❓ "Code won't run"
→ Click on the cell first (blue border), then Shift + Enter

❓ "Variable undefined error"
→ Run cells from top to bottom in order

❓ "Notebook seems frozen"
→ Go to Kernel menu → Restart Kernel

❓ "Can't connect to database"
→ Make sure Docker is running: docker-compose up -d
